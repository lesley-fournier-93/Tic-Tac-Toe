# ❌⭕ Tic Tac Toe

A simple Tic Tac Toe game built with HTML, CSS, and JavaScript.

## 🚀 Features
- Interactive game board
- Two-player gameplay
- Win and draw detection
- Simple and clean user interface

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript

## 🎯 Purpose
This project was created to practice JavaScript logic, DOM manipulation, and interactive frontend development.
